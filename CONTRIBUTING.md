# Contributing Guidelines

Thank you for interest in contributing to **Molavi Agent Skills**! 

To maintain the high quality, safety, and security of this collection, please follow these guidelines when contributing:

## Submission Rules

1. **Deduplication Check**: Before submitting a new skill, search the `skills/` directory to ensure a similar skill does not already exist.
2. **Secrets & Privacy**: Ensure the files contain no credentials, private API keys, local paths, or sensitive project information. All examples should use clean, secure, and mock placeholders.
3. **English Documentation**: Write all `SKILL.md` configurations, guides, and comments in English.
4. **Structured Format**: Follow the standard `SKILL.md` frontmatter layout (Name, Description, and detailed rules).

## How to Submit

1. Fork the repository on GitHub.
2. Clone your fork locally and create a new feature branch.
3. Stage and commit your changes with clear descriptions.
4. Open a Pull Request pointing to the main branch of `tmolavi/antigravity-awesome-skills`.
